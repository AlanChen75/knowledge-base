## 專案概覽

Dawarich 是開源自架位置歷史管理平台，Google Timeline 的替代方案。Ruby on Rails 8.0 + PostgreSQL/PostGIS。開發者 Evgenii Burmakin（GitHub: Freika）。

- GitHub Stars：8.7k
- 授權：AGPL-3.0
- 最新版本：v1.6.1（2026-04-01）
- 2025 年發佈 77 個版本，開發非常活躍
- 官方：https://dawarich.app / https://github.com/Freika/dawarich

## 核心功能

- **地圖視覺化**：互動式地圖 + 日期範圍篩選 + 熱力圖 + 路線追蹤
- **旅行統計**：到訪國家/城市、移動距離、停留時間，按年/月拆分
- **家庭共享**：雙方同意後可互相查看位置
- **照片整合**：Immich / PhotoPrism 整合，地圖上顯示照片拍攝位置（不複製照片）
- **資料匯入**：Google Maps Timeline、GPX、GeoJSON、OwnTracks 等

## 部署

Docker Compose 一鍵部署：

```bash
docker compose -f docker/docker-compose.yml up
```

容器：Rails App + PostgreSQL/PostGIS + Redis。預設 CPU 0.50 / RAM 4G。支援 Linux、macOS、Windows、Synology NAS。

## 競品比較

| 面向 | Dawarich | OwnTracks | Traccar |
|------|----------|-----------|---------|
| 定位 | 個人位置歷史管理 | 位置數據收集 | 車隊/裝置追蹤 |
| Web UI | 完整互動式地圖+統計 | 無內建 | 車隊管理風格 |
| 照片整合 | Immich/PhotoPrism | 無 | 無 |
| 資料匯入 | Google Timeline/GPX/GeoJSON | 無 | 有限 |

關鍵：Dawarich 支援以 OwnTracks 為手機端，自己做後端 + 視覺化。已在用 OwnTracks 可無痛遷移。

## Immich 整合

設定頁面填 Immich URL + API Key（需 `asset.read` + `asset.view` 權限）。照片自動出現在地圖對應位置，搭配旅行記錄形成時空日記。照片留在 Immich，不複製。

## API 與自動化

REST API（`/api-docs`），支援批次推送位置點。社群有 Python 客戶端（`dawarich-api` PyPI）和 Home Assistant 整合。Webhook 尚未實作，API 輪詢可替代。

## 隱私與安全

所有資料存在自己的 PostgreSQL，不經第三方。可匯出 GPX/GeoJSON。AGPL-3.0 確保永遠開源。建議搭配反向代理 + HTTPS + 強密碼。預設帳密 `demo@dawarich.app / password` 部署後必須改。

## 應用場景

### 與 Glance Dashboard 整合

Dawarich 跑在 `localhost:3000`，可作為 Glance iframe widget 或透過 API 拉統計數據嵌入。

### 與 SecondBrain 結合思路

1. **會議地點自動記錄**：API 查詢特定日期位置，自動標記「今天去了哪裡開會」
2. **出差報告自動化**：匯出旅行統計，自動生成出差摘要
3. **時空知識圖譜**：Dawarich（位置）+ Immich（照片）+ SecondBrain（筆記）三合一
4. **地點觸發筆記**：等 webhook 實作後，進入特定地點自動建立會議記錄模板

### 商業趨勢

Google 2024 年關閉線上版 Maps Timeline，推動自架位置工具需求爆發。位置隱私是數位主權（Digital Sovereignty）運動核心議題，與 Immich（照片）、Obsidian（筆記）形成個人資料自主生態。

## 結論

**推薦部署**。目前自架位置歷史管理中最成熟、UI 最好、生態最完整的選項。

部署順序：Docker Compose → 匯入 Google Timeline 歷史資料 → 連接 Immich → 設定手機追蹤 → 探索 API 整合 SecondBrain

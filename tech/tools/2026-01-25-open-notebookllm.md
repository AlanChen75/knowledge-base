# Open NotebookLM - 開源版 NotebookLM 複刻專案

## 📊 元資訊
- **難度**：⭐⭐⭐⭐
- **來源類型**：GitHub 開源專案
- **作者**：阿亮老師 (Teacher Liang)
- **筆記時間**：2026-01-25 18:52

## 📌 摘要
Open NotebookLM 是一個開源的 NotebookLM 複刻專案，支援多種 AI 提供商（Gemini、OpenAI、Anthropic、Ollama 等），具備 RAG 檢索、Podcast 生成、語音轉文字等功能。可以上傳多種格式文件進行智能問答，並自動生成學習內容。

## 🏷️ 標籤分類
- **大分類**：tech
- **小分類**：tools / AI 應用

## 🔑 關鍵要點
1. **多 AI 提供商支援**：Gemini、OpenAI、Anthropic、Ollama、Groq、DeepSeek
2. **多格式文件支援**：PDF、Word、Excel、網頁、YouTube、音訊檔案
3. **Podcast 生成**：多講者對話 + TTS 語音合成
4. **Studio 功能**：自動生成摘要、心智圖、閃卡、測驗、簡報、資訊圖表
5. **進階搜尋**：RRF 融合、查詢擴展、LLM 重新排序
6. **技術架構**：Flask + React + TypeScript + TailwindCSS

## 🧠 概念連結
- **RAG (Retrieval-Augmented Generation)**：結合檢索與生成的 AI 技術
- **NotebookLM**：Google 的 AI 筆記工具，可分析文件並生成 Podcast
- **TTS (Text-to-Speech)**：文字轉語音技術
- **Bloom's Taxonomy**：布魯姆教育目標分類學，用於智能學習內容

## 💡 與我的連結
這個專案可以用來：
- 建立個人知識庫的進階版本，支援多格式文件分析
- 自動將筆記或文件轉換成 Podcast 收聽
- 結合 RAG 技術提升知識檢索效率
- 作為學習 Flask + React 全端開發的參考

## 📝 我的註解與思考
這是一個功能非常完整的 NotebookLM 替代方案。特別是支援多種 AI 提供商這點很實用，可以根據需求選擇不同的 LLM。Podcast 生成功能也很有趣，可以把文字內容轉換成可以邊走邊聽的音訊。

技術架構使用 Flask + React 是主流選擇，值得學習其 RAG 實作方式。

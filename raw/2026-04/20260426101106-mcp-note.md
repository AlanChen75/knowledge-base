# Anthropic Project Deal：AI 代理人市集實驗

> **可作為課程教材引用**
> 適用課程：AI 四級課程、AI 永續兩級課程、AI 智慧製造三級課程、企業培訓（AI 轉型、ESG 治理）
> 引用時請使用原始來源，避免使用網路上的中文加工版本（多有戲劇化誇大）

## 📌 摘要

Anthropic 於 2025 年 12 月在舊金山辦公室執行為期一週的內部實驗 Project Deal，由 69 名員工各獲 100 美元預算，全程由 Claude 代理人在 Slack 自主議價、買賣實體物品。實驗共完成 186 筆交易、總值逾 4,000 美元。**核心發現**：當代理人由較強模型（Opus 4.5）代表時可獲得客觀上更佳結果，但弱模型（Haiku 4.5）使用者完全察覺不到自己吃虧——形成一種「隱形不平等」（agent quality gap）。報告於 2026 年 4 月 24 日公開發布。

## 🔑 關鍵要點

### 實驗設計
1. **樣本**：69 名 Anthropic 舊金山辦公室自願員工（self-selected pool），每人 100 美元禮品卡預算
2. **流程**：Claude 先訪談每位員工（想賣什麼、想買什麼、議價風格偏好），轉成各自的 system prompt，再放入 Slack 頻道；實驗期間人類完全不介入
3. **四個平行市集**：
   - Run A、Run D：全員 Opus 4.5
   - Run B、Run C：50/50 隨機分配 Opus 4.5 或 Haiku 4.5
   - 只有一個是「真實 run」，會實際交付物品（事後才告知參與者）
4. **物品**：真實員工二手物，從滑雪板到一袋乒乓球

### 量化結果（重要數據，引用時務必使用這組）
| 指標 | 數值 |
|------|------|
| 總交易筆數 | 186 |
| 總交易金額 | 約 $4,000 |
| 平均成交價 | $20.05 |
| 中位成交價 | $12 |
| Opus 賣家平均多賺 | +$2.68 |
| Opus 買家平均少付 | -$2.45 |
| Opus 賣家 vs Haiku 買家平均成交價 | $24.18 |
| Opus vs Opus 平均成交價 | $18.63 |
| Opus 用戶 fairness 評分（1-7） | 4.05 |
| Haiku 用戶 fairness 評分（1-7） | 4.06 |
| 願意付費使用此服務的參與者比例 | 46% |

### 三大核心發現
1. **代理人市集是可行的**：AI 完全自主完成從掛單、報價、討價還價到成交的全流程，無需預設議價協議
2. **模型能力差距會直接轉化為金錢差距**：但差距是「平均 $2-3 美元」級別（約 13-30%），不是某些中文加工版宣稱的 70%
3. **弱模型用戶察覺不到劣勢**：Haiku 用戶與 Opus 用戶的滿意度、公平感評分幾乎相同（4.06 vs 4.05）——這是最值得擔憂的發現

### Prompt Engineering 的角色
- 議價風格指令（如「用窮途末路牛仔的口吻」、「強硬議價並先低報」）對最終成交價無統計顯著影響
- 「強硬賣家」確實得到較高價格，但只是因為他們開價就高
- **結論需謹慎解讀**：這不代表所有 prompt engineering 都無效，system prompt 仍是把使用者偏好編碼進代理人的必要環節；只是「議價語氣」這個維度被模型實力差距吸收掉了

## 🧠 概念連結

- **Agent Economy（代理人經濟）**：AI 代表人類執行交易的經濟模式正在從理論走向實作
- **Information Asymmetry（資訊不對稱）**：傳統由雙方資訊差距導致，AI 時代轉為由「代理人能力差距」導致
- **Project Vend（前作）**：Anthropic 之前讓 Claude 經營辦公室小店的實驗，是 Project Deal 的前身
- **Harness Engineering**：與我目前在做的 agent 編排架構直接相關——代理人的「外殼」（system prompt + 上下文 + 工具）會決定它代表使用者的能力上限
- **Disclosure as Regulation**：Anthropic 自己呼籲的監管方向是「揭露對方使用什麼等級的代理人」，而非禁止使用代理人

## 💡 與我的連結

### 對課程教材的價值
1. **AI 四級課程**：可作為「AI 代理人未來」單元的核心案例，搭配 Project Vend 形成連續案例組
2. **AI 永續兩級課程**：從 ESG/治理視角切入，討論「演算法公平性」與「數位包容性」的新議題
3. **AI 智慧製造三級課程**：類比到 B2B 採購場景——當供應商與買方都用 AI 代理人議價，模型差距如何影響供應鏈經濟
4. **企業培訓（政府/企業客戶）**：建議切角是「警示性案例」+「監管前瞻」，**不要使用「贏家通吃」「修羅場」這類網路加工論述**——政府客戶會懷疑學術嚴謹性

### 對自己研究工作的啟示
1. **HTF-CNN 論文同儕審查、citation verification、expert review matrix 的工具鏈**：模型實力差距會直接影響輸出品質，且**自己看不出來**——這意味著關鍵環節應使用 Opus 級模型，不應為了省 token 用低階模型
2. **七維度專家審查矩陣**：跑在不同模型上會產出不同深度的審查，但表面看都「合理」——應該設計交叉驗證機制
3. **Context gap 問題（Claude Code vs Claude.ai）**：這個實驗變相證實了「代理人的能力 = 模型 × 上下文完整度」，缺一不可

### 對網站基礎設施的策略意義
- Ai100講、永續100講、保險資訊全球站作為 RAG 內容來源，未來真正的價值不是「人類讀者瀏覽」，而是「成為其他 AI agent 在做研究/採購時的引用來源」
- 這呼應 Anthropic 提到的「Optimizing for AI agent attention could become a powerful tool」

## 📝 我的註解與思考

**這個實驗最被低估的發現不是「強模型贏」，而是「弱模型用戶完全沒感覺」**。Fairness 評分 4.06 vs 4.05 這組數字的意義是：當你被宰時，你的主觀體驗依然是「公平」。這在台灣的 AI 普及論述裡幾乎沒有被討論——大家還在討論「該不該用 AI」、「AI 會不會取代我」，但下一階段的真議題是「我的代理人是不是被別人的代理人吃掉了，而我永遠不會知道」。

**對中文 AI 社群的觀察**：這個案例在被翻譯傳播時，常被加工成「免費仔被摑一巴」「贏家通吃修羅場」的消費焦慮論述，把 70% 溢價之類的杜撰具體例子（單車 65 vs 38 美元）夾雜在真數據中。這種寫法在社群媒體會獲得高轉發，但**進入課程或學術語境會傷害可信度**。引用時務必回到原始來源。

**與我「不要做免費仔」的思考線一致**：但我會把它精緻化為——不是「窮人用不起 AI」，而是「在你的關鍵決策節點（合約、採購、論文投稿、商業談判），代理人實力差距會無聲無息累積成系統性劣勢」。這比情緒化口號更有教學價值。

**研究方法上值得學習的地方**：Anthropic 用「四個平行市集 + 一個真實 run」的設計同時取得真實激勵與對照實驗——這是很巧妙的研究設計，值得我在自己的 NILM 實驗設計（特別是 ablation study）參考。

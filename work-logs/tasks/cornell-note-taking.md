## 研究重點

### 1. 康乃爾筆記法核心結構
- **筆記欄（Notes Column）**: 右側，記錄主要內容
- **線索欄（Cue Column）**: 左側，關鍵字、問題、提示
- **摘要欄（Summary Section）**: 底部，2-3 句總結
- **標題區（Header）**: 主題、日期、來源

### 2. 使用流程
1. **記錄（Record）**: 課堂/閱讀時在筆記欄記錄
2. **提煉（Reduce）**: 事後在線索欄寫關鍵字與問題
3. **背誦（Recite）**: 遮住筆記欄，用線索欄自我測驗
4. **反思（Reflect）**: 思考內容意義與應用
5. **複習（Review）**: 定期回顧摘要與線索

### 3. Markdown 模板設計考量
- 如何用 Markdown 語法呈現三欄結構？
- 是否使用表格？或分區塊？
- Frontmatter 如何整合？
- 如何支援標籤與分類？

### 4. 整合到知識庫

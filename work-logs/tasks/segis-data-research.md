# SEGIS 資料集研究與下載

## 目標
從內政部 SEGIS 平台下載關鍵資料集，用於分析台灣地區發展趨勢、產業脈絡、房地產發展。

## 平台資訊
- **網址**: https://segis.moi.gov.tw/STATCloud/Catalog
- **下載方式**: ZIP 檔（內含 CSV + Schema.ini + Info.ini）
- **下載連結格式**: `https://segis.moi.gov.tw/STATCloud/reqcontroller.file?method=filedown.downloadproductfile&code=XXX&STTIME=114Y12M&STUNIT=U01CO&BOUNDARY=全國`
- **需要登入**: 部分資料集需會員帳號

## 優先下載清單

### 第一優先：地區發展趨勢
- [ ] 各級統計區人口統計（人口流動方向）
- [ ] 三段年齡組人口 0-14/15-64/65+（高齡化程度）
- [ ] 人口消長 — 遷入/遷出（縣市吸引力）
- [ ] 教育程度別人口（知識經濟熱區）
- [ ] 出生/死亡統計（自然增減率）
- [ ] 外來人口/新住民統計（國際化程度）

### 第二優先：產業脈絡
- [ ] 勞動力參與率
- [ ] 各業就業人數（產業集中度）
- [ ] 失業率（年齡/教育別）
- [ ] 外籍勞工統計（製造業/看護密度）
- [ ] 職業工會/勞資爭議
- [ ] 稅收實徵淨額（各縣市經濟活力）

### 第三優先：房地產發展
- [ ] 已登記土地面積（住宅/商業/工業用地分布）
- [ ] 市地重劃辦理面積（未來發展熱區）
- [ ] 都市更新地區
- [ ] 戶數/戶量統計（小宅需求趨勢）
- [ ] 人口密度（開發飽和度）

## 黃金交叉分析組合
1. 人口遷入遷出 + 三段年齡組 → 地區成長與人口結構
2. 稅收實徵淨額 → 縣市經濟實力
3. 各業就業人數 + 外籍勞工 → 產業結構
4. 已登記土地面積 + 市地重劃 + 都市更新 → 房地產供給面
5. 戶數統計 + 人口密度 → 房地產需求面
